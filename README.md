# 🎨 AI Crafter Toolkit

Ce dépôt sert de base pour créer, organiser et tester des agents, skills et templates dédiés au développement d’outils pour Claude Code et Github Copilot.

## Description

Vous y trouverez :

- Des agents et skills prêts à l’emploi ou personnalisables
- Des templates pour faciliter la création de nouveaux outils (skills, agents, hooks, etc.)
- Des ressources pour enrichir et étendre les capacités de Claude Code et Copilot

L’objectif est de centraliser et accélérer le développement d’extensions pour ces plateformes.
